## Baseline Implications

The most important baseline to beat for this project is not only shortest path. A research-defensible evaluation should compare against:

- Random ranking.
- Shortest-distance ranking.
- Prompt/SBERT ranking when preference text exists.
- Profile-only ranking.
- Hybrid prompt/profile ranking.
- If data volume allows, a learned ranking baseline such as LightGBM.

The **Personalized Route Recommendation Based on User Habits for Vehicle Navigation** paper is the closest conceptual baseline because it combines user habits, route features, route ranking, and comparisons against non-personalized and learned baselines. For the current prototype, the honest target is to show that pseudo-history profile and hybrid ranking improve over random and shortest-distance ranking under the same candidate sets and feature-similarity relevance definition.
