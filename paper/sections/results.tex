\subsection{Trace Reconstruction}
The Toronto OSM trace probe currently contains $15{,}000$ parsed trackpoints with timestamp coverage $1.00$. The pipeline found $6$ pseudo-segments and $6$ useful segments, for a useful segment rate of $1.00$. Approximate map matching succeeded for all attempted useful segments. The median GPS-to-route median distance is approximately $7.17$ meters, indicating close geometric alignment between sampled points and reconstructed routes. However, the median route-distance ratio is approximately $2.15$, above the strict target ceiling of $2.0$.

Therefore, the current strict prototype usability flag is false. The exploratory usability flag is true under the looser $2.5$ route-distance-ratio ceiling. This distinction is central: the data is useful for exploratory pseudo-history experiments, but the reconstruction is not yet clean enough for strong claims about route fidelity.
